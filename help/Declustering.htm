Declustering an Earthquake Catalog

 

 

Introduction

 

To separate the dependent from the independent seismicity, the earthquake catalog can be declustered. The decluster algorithm used in ZMAP is based on an algorithm developed by Reasenberg (Reasenberg, 1985). Most of the ZMAP software for the declustering has been developed by Alexander Allmann. The Y2K compatible FORTRAN code was provided by C. Goltz (Univ. Kiel).

 

When you choose "Decluster the catalog" from the "Tools" pulldown menu of either the map or cumulative number windows, an input parameter box

will appear. Eight different parameters can be set:

 

·        taumin: the look-ahead time for un-clustered events in days (default is one day).

·        taumax: the maximum look-ahead time for clustered events in days (default 10 days)

·        P1: a measure of the confidence that you are observing the next event in the sequence (default is 0.95)

·        xk: the factor used in xmeff (default 0.5)

·        xmeff: the "effective" lower magnitude cutoff for the catalog.

·        rfact: the factor for the interaction radius of dependent events (default is 10)

·        Epicenter-Error: horizontal location error in km (default is 1.5)

·        Depth-Error: vertical error in hypocenter location in km (default is 2) For additional information about these parameters hit "Info".

 

 

In ZMAP6, there are two option to decluster: within Matlab, or using an external call. The main advantage to the second option is that it is much faster, and provides access to the original output files. In either option, declustering can be memory consuming for larger catalogs.

 

In order to use the FORTAN option, you need to be able to execute the file ./external/clus.f . The PC version includes a precompiled clus.exe. On UNIX/LINUX systems, you may have to compile this file. To run the declustering, the data is saved in the appropriate format in the file ./external/data and the program called with a ‘Unix’ call from within Matlab. Output data are the files ./external outf.clu, outf.dec, and outf.out.
