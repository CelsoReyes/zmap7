Plotting Topography Data in ZMAP

 

Last updated: 11/2001 sw

 

Topography and bathymetry data make an epicenter maps much more vivid. However, plotting these maps can be a considerable drain on your computer power – a good graphics cards is necessary once you like to attempt to plot detailed topo maps. ZMAP allows plotting of the following DEM (Digital Elevation Model) data sets:

  • Terrain Base, a 5 min global topography and bathymetry. This data set is available for download from The Mathworks web site at ftp://ftp.mathworks.com/pub/mathworks/toolbox/map/extMapData/tbase The figure below shows an example for Japan and Korea. The untared data should be moved into the directory zmap/dem. To read the data file tbase.bin, the mapping tool function tbase is used, which is only available if you have purchased a license of the mapping Toolbox. Below is an example of the world plus plate boundaries from this database.

 

  • ETOPO2, a 2 min global topography and bathymetry data set (Sandwick and Smith, 136 MB file size). If you Internet connection is fast enough, you can download the file  topo_6.2.img, and copy it into the zmap/dem directory. The Mapping Toolbox is NOT needed to load this data. Both data files, ETOPO2 and Terrain Base are included on the  ZMAP CD-ROM. Once you have copied this data set into the dem directory, plotting DEM data is easy: Simply select the "Plot Topography Map > 2 deg Resolution (Etopo2)"  from the Tools menu of the seismicity map. The area covered will be equal to the area currently show in the seismicity map. Zoom into the seismicity map to plot a smaller region. Below an example for the western USA. The right frame shows the same map plotted using the mapping toolbox in a 3D rendering. This plot can be achieved using the “plot using mapping toolbox’ button on the lower left side of the topography map.

  • 30 arc seconds land topography: GTOPO30 and GLOBE DEM. These DEM’s have a 1 x 1 km resolution of the landmass of the Earth. The data is saved in a number of tiles, which can be downloaded from the Internet for free. See http://edcwww.cr.usgs.gov/landdaac/gtopo30/gtopo30.html and http://www.ngdc.noaa.gov/seg/topo/globe.shtml for more details and download. Each globedem files is about 120 Mb. An example for Switzerland is shown below.

 

To make ZMAP work with GLOBE 1 DEM files, you need to create a directory structure underneath the dem  directory. First create a director ./dedm/esri, the copy the subdirectory  ftp://ftp.ngdc.noaa.gov/GLOBE_DEM/data/elev/esri/hdr into esri. Next download the necessary tiles and rename them by deleting the .gz extension. You do not need to unzip them (at least I did not need to do this on my PC).

To use Gtopo30 files, download the necessary tiles and uncompress them.
